# SQL Sales Analysis Projekt

## Übersicht

Dieses Projekt demonstriert SQL-Datenanalysekenntnisse mit PostgreSQL.

Ziel des Projekts ist die Analyse von Verkaufsdaten, die Berechnung von Umsatzkennzahlen sowie die Anwendung wichtiger SQL-Konzepte, die in der Datenanalyse verwendet werden.

Das Projekt beinhaltet:

- Erstellung einer Datenbankstruktur (Database Schema)
- Einfügen von Beispieldaten (Sample Sales Data)
- SQL-Abfragen zur Datenanalyse
- JOINs
- GROUP BY
- HAVING
- Subqueries
- Window Functions

## Dateien

### schema.sql

Enthält:

- CREATE TABLE-Anweisungen
- Tabellenstruktur für Kunden, Produkte und Bestellungen
- Beispiel-Datensätze für die Analyse

### analysis_queries.sql

Enthält:

- Kundenanalysen
- Umsatzberechnungen
- Produktanalysen
- Ranking-Abfragen
- Aggregationsfunktionen
- Beispiele für JOINs und GROUP BY
- Beispiele für Window Functions
