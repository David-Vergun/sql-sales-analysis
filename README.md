# SQL Sales Analysis Project

This project demonstrates SQL data analysis skills using PostgreSQL.

The project includes:

- Database schema creation
- Sample sales data
- SQL analysis queries
- JOINs
- GROUP BY
- HAVING
- Subqueries
- Window Functions
